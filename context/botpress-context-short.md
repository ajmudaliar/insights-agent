# Botpress Domain Context (Short)

Botpress is an AI-powered chatbot and agent building platform. It enables users to create conversational AI assistants through a visual flow editor (Studio), deploy them across multiple channels (WhatsApp, Slack, Messenger, Telegram, web), and manage them via a Dashboard.

**Core Components:**
- **Studio**: Visual bot builder with drag-and-drop flow editor, prompt management, and code editor
- **Dashboard**: Bot management, analytics, workspace administration
- **Webchat**: Embeddable chat widget for websites
- **Integrations**: 64+ pre-built connectors (Slack, Teams, Salesforce, HubSpot, Notion, etc.)

**Key Features:**
- Autonomous AI agents powered by LLMs (OpenAI, Anthropic, Google AI)
- Knowledge base integration for RAG-based responses
- Human-in-the-loop (HITL) for agent handoff
- Multi-channel deployment
- Tables for data storage
- Workflows for complex conversation flows

**User Types:**
- Developers building custom bots via SDK/CLI
- Business users creating bots visually in Studio
- Enterprise teams managing multiple bots across workspaces

**Pricing:** Freemium with pay-as-you-go scaling.
